# Pizza Delivery Website - Oasis Infobyte Internship

This project is a Pizza Delivery Website created as part of my Web Development and Designing Internship at Oasis Infobyte.

## Features

- Navigation bar with multiple sections
- Hero section with order button
- Pizza menu display
- Order section with input fields
- Footer with project details
- Clean and simple design using CSS

## Technologies Used

- HTML
- CSS

## Project Structure

- index.html → Website structure
- style.css → Website styling

## Project Purpose

This project was built to understand how to create a multi-section website layout using HTML and CSS.
